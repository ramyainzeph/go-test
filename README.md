# Docker Go Test

- This is a simple Go application that prints "Docker Go test working!" inside a Docker container. 
- The application demonstrates how to set up a Go project inside a Docker container.


## Steps to Build and Run the Dockerized Go Application

### 1. Clone the Project
### 2. Install Docker   *(already installed skip this step)
### 3. cd docker-go-test
### 4. docker build -t docker-go-test .
### 5. docker run --rm docker-go-test    * (run the Go script)

  ###  Docker Go test working!  

### 6.docker rmi docker-go-test   * Clean Up (Optional)
